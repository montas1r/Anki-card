# Anki Flashcards WebApp

A spaced-repetition flashcard app (Anki-style) with decks, built with Flask + Vanilla JS.

## Features
- Create/delete decks
- Add/delete cards
- Study mode with SM-2 spaced repetition
- Difficulty ratings (Again / Hard / Good / Easy)

## Setup

```bash
cd anki-webapp/backend
pip install -r requirements.txt
python app.py
